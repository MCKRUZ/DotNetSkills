# EY Brand Guidelines for PowerPoint Presentations

## Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| EY Navy | #1E1847 | RGB(30, 24, 71) | Headlines, primary text, emphasis |
| EY Yellow | #FFDD00 | RGB(255, 221, 0) | Accents, highlights, key insights (use sparingly) |
| Black | #000000 | RGB(0, 0, 0) | Body text, secondary content |
| White | #FFFFFF | RGB(255, 255, 255) | Backgrounds, negative space |

### Secondary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| Light Gray | #E6E6E6 | RGB(230, 230, 230) | Subtle backgrounds, dividers |
| Medium Gray | #999999 | RGB(153, 153, 153) | Secondary text, captions |
| Dark Gray | #333333 | RGB(51, 51, 51) | Alternative to black |

### Data Visualization Colors

**Sequential (single metric):**
1. EY Navy (#1E1847)
2. Medium Gray (#999999)
3. Light Gray (#E6E6E6)

**Categorical (multiple items):**
1. EY Navy (#1E1847)
2. Dark Gray (#333333)
3. Medium Gray (#999999)
4. Light Gray (#E6E6E6)
5. Black (#000000)

**Highlight:**
- Use EY Yellow (#FFDD00) only for the most important data point
- Never use yellow for text (readability issues)
- Yellow should be used for < 10% of visual elements

## Typography

### Font Families

**EY Sans** (Primary - modern, clean)
- Use for: Body text, subheads, captions
- Available weights: Light, Regular, Medium, Bold
- Similar alternatives: Arial, Helvetica, Open Sans

**EY Serif** (Secondary - sophisticated, authoritative)
- Use for: Main headlines, section titles
- Available weights: Regular, Bold
- Similar alternatives: Georgia, Times New Roman

### Font Sizes

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Slide Title | EY Serif | 40-54pt | Bold |
| Section Headline | EY Sans | 30-36pt | Medium |
| Subhead | EY Sans | 24-30pt | Medium |
| Body Text | EY Sans | 18-22pt | Regular |
| Caption | EY Sans | 14-16pt | Light |
| Footer | EY Sans | 10-12pt | Light |

### Text Guidelines

- **Sentence case** for all headlines (not Title Case)
- Minimum 18pt body text for readability
- 1.2-1.5 line spacing for body text
- Maximum 6 bullets per slide
- Maximum 6-8 words per bullet

## Logo Usage

### Logo Placement

**Title Slide:**
- Logo: Top-left or top-center
- Size: ~1 inch width
- Clear space: 0.5x logo height on all sides

**Content Slides:**
- Logo: Bottom-right or top-right (consistent throughout)
- Size: ~0.5 inch width
- Placement: Same position on every slide

### Logo Rules

- **Never** stretch or distort the logo
- **Never** place logo on photographic backgrounds
- **Never** alter logo colors
- **Always** maintain clear space around logo
- **Use** high-resolution logo files (300+ DPI)

## Slide Layout

### Standard Layouts

#### Headline-Body-Body-Body
```
+----------------------------------+
| ASSERTIVE HEADLINE (40-54pt)     |
+----------------------------------+
|                                  |
| • Supporting point one           |
|   with additional detail         |
|                                  |
| • Supporting point two           |
|   with data or evidence          |
|                                  |
| • Supporting point three         |
|   with call to action            |
|                                  |
+----------------------------------+
```

#### Headline-Graphic-Support
```
+----------------------------------+
| ASSERTIVE HEADLINE               |
+----------------------------------+
|                                  |
|       [VISUAL ELEMENT]           |
|       (Chart, Diagram, Photo)    |
|                                  |
| Supporting insight or            |
| commentary on the visual         |
|                                  |
+----------------------------------+
```

#### Headline-Data-Insight
```
+----------------------------------+
| ASSERTIVE HEADLINE               |
+----------------------------------+
|                                  |
|      [CHART OR GRAPH]            |
|      With highlighted data       |
|                                  |
| Key insight: The data shows      |
| a 40% improvement driven by      |
| the new initiative               |
|                                  |
+----------------------------------+
```

### Margins and Spacing

- **Slide margins**: 0.5 inch on all sides
- **Content margins**: 0.75 inch from slide edge
- **Spacing between sections**: 0.25 inch minimum
- **Spacing before headline**: 0.25 inch
- **Spacing after headline**: 0.25 inch

## Slide Master Templates

### Title Slide Master
- Background: White or very light gray
- Headline: Centered, 54pt EY Serif Bold
- Subtitle: Centered, 30pt EY Sans Medium
- Logo: Top-center, 1 inch width
- Accent line: 3pt EY Navy horizontal rule

### Section Divider Master
- Background: EY Navy (#1E1847)
- Section title: Centered, 48pt EY Serif Bold, White
- Optional: Subtitle in EY Sans Medium, EY Yellow

### Content Slide Master
- Background: White
- Headline: Left-aligned, 44pt EY Serif Bold, EY Navy
- Body: Left-aligned, 20pt EY Sans Regular, Black
- Logo: Bottom-right, 0.5 inch width
- Footer: Slide number, optional date

## Visual Elements

### Icons
- Use consistent icon style throughout
- Prefer line icons over filled icons
- Use EY Navy for primary icons
- Use EY Yellow only for most important icon

### Charts
- Chart title = assertive headline
- Remove chart borders and backgrounds
- Use gridlines sparingly (horizontal only, light gray)
- Highlight key data point with EY Yellow
- Label axes clearly
- Include data labels for key values

### Diagrams
- Use 2-3 colors maximum
- EY Navy for primary elements
- Gray for secondary elements
- Yellow for highlights
- Clean lines, no drop shadows
- Consistent stroke weight

### Photography
- High resolution (300 DPI minimum)
- Professional quality
- Relevant to content
- Full-bleed or framed with white space
- Avoid stock photo clichés

### ey-dark.potx

## Accessibility

- Minimum contrast ratio: 4.5:1 for body text
- Minimum font size: 18pt
- Don't rely on color alone to convey meaning
- Provide text descriptions for visuals
- Test in actual presentation environment

## File Naming Convention

- Use underscores, not spaces: `EY_Executive_Summary.pptx`
- Include version: `EY_Executive_Summary_v2.pptx`
- Include date: `EY_Executive_Summary_20250107.pptx`
